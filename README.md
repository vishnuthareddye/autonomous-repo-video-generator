# Autonomous Repo-to-Video Creator Agent

A highly unique, experimental AI engine pipeline that translates structured codebase systems into interactive audio-visual video assets. It parses files, utilizes local LLMs to script summaries, and generates synchronized voice track assets autonomously.

## 🌟 Strategic Core Features
* **Automatic Code Scoping**: Traverses file trees to identify main software architectures.
* **Local Narrative Processing**: Leverages private Ollama instance layers to eliminate video script synthesis data transit overheads.
* **Integrated Audio Assembly**: Transforms raw logic syntax straight into standard spoken mp3 audio payloads.

## 🛠️ System Tech Stack
* **Language System**: Python 3
* **AI Processing Interface**: Ollama Framework Engine (Llama 3)
* **Speech Synthesis Engine**: Google Text-to-Speech Core Array (`gTTS`)

## 🚀 Execution Guide
1. Run your local model instance: `ollama run llama3`
2. Install library parameters: `pip install -r requirements.txt`
3. Execute the media compilation sequence: `python generator.py`
